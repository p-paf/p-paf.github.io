Course Description

Python is one of the most popular programming languages of the 21st century. Functional programming (FP) is a way of thinking about software construction that emphasizes functional purity and immutability. Demand for fp is steadily increasing. According to Stack OverFlow Developer Survey 2019, two of highest paying programming languages emphasize the functional approach. The course covers the fundamental concepts of fp and make copious use of Python and idioms to create beautiful programs.
